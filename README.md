# FaceCheck Reverse Image Search - Firefox Extension

## Overview

FaceCheck Reverse Image Search is a browser extension that enables you to search the internet by face. Simply right-click on any image of a person to find almost anyone online using advanced facial recognition technology.

## Features

- **Easy Face Search**: Right-click on any image of a person to instantly search for that individual
- **Direct Integration**: Seamlessly connects with FaceCheck.ID's facial recognition search engine
- **Browser Action**: Quick access via browser toolbar icon with usage instructions
- **Privacy-Focused**: No data collection permissions required
- **Cross-Platform**: Available for Firefox (Gecko-based browsers)

## Installation

1. Download and install the extension from the appropriate browser store or here on github (.xpi, view raw).
2. The extension icon will appear in your browser toolbar
3. No additional setup required

## How to Use

### Method 1: Context Menu (Recommended)
1. **Navigate** to any webpage containing an image of a person
2. **Right-click** on the image
3. Select **"Find Person with FaceCheck.ID"** from the context menu
4. A new tab will open with search results from FaceCheck.ID

This was originally made in chrome However I've figured I'd port it to firefox users.
